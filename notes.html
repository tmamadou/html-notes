Overflow - (CSS)

overflow: visible; - when text in a border doesn't fit, it will show it anyways

overflow: hidden; - it will hide that text that doesn't fit in the order

overflow: scroll; - will add a scroll wheel so you can scroll to the rest of the text inside of the border

overflow: auto; - will automatically add a scroll wheel if its needed.
